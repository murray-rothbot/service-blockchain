## Description

This is a NestJS APIs to provide BTC exchange rates.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Extras

```bash
# remove all deps and reinstall from scratch
$ npm run clean

# check for dependencies updates
$ npm run check
```
